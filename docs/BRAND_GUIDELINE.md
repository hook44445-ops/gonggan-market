# 공간마켓 브랜드 가이드라인 (Brand Message System v1.4 · Final)

> 이 문서는 UI/UX 및 카피 작업 시 참조하는 브랜드 원칙이다. 기능/구조 변경 지시가
> 아니다 — 화면별 구체적인 수정은 별도 작업 지시로 진행하며, 이 문서는 그 판단 기준이다.

## 1. 브랜드 철학 (변경 금지)

다음 네 가지는 공간마켓의 핵심 아이덴티티이며, 앞으로도 임의로 변경하지 않는다.

- Deep Green — `C.brand #2E5F4B` / `C.brandD #1D3D2F` 계열 (진녹색)
- Gold — `C.gold #C8A15A` 계열
- "좋은 공간과 좋은 이야기가 모이는 곳"
- "사람을 믿는 거래가 아니라 기록을 믿는 거래"

## 2. 3 Layer Tone System

화면 성격에 따라 톤을 구분한다.

| Layer | 대상 화면 | 톤 |
|---|---|---|
| **Trust Mode** | 계약·견적·에스크로·결제·분쟁·GPS·프로젝트 진행 | 가장 정확하고 담백한 톤. 유머 금지 |
| **Friendly Mode** | 홈·온보딩·마이페이지·업체 소개·후기·채팅 | 카카오처럼 친근하고 따뜻한 톤 |
| **Lounge Mode** | 라운지·댓글·좋아요·빈 화면·완료 화면·커뮤니티 알림·에러 메시지 | 생활감 있는 친근한 UX, 가벼운 위트 허용 |

## 3. 내부 운영 원칙 — 외부 문서 노출 금지

아래는 팀 내부에서 톤을 설명하기 위한 운영 키워드다. 외부 문서·브랜드 소개·사용자
노출 문구에는 사용하지 않는다.

| 내부 운영 키워드 (팀 내부 전용) | 외부 표현 (문서/카피에 사용) |
|---|---|
| 병맛 감성 | 생활형 유머 |
| B급 감성 | 가벼운 위트 / 친근한 UX |
| 피식 웃는 유머 | 따뜻한 카피 / 카카오 같은 친근함 |

## 4. 병맛은 브랜드가 아니라 UX 장치

병맛/위트는 브랜드 정체성이 아니라 특정 화면에서만 쓰는 UX 장치다. 적용 위치를
엄격히 구분한다.

- **허용**: 라운지, 빈 화면, 완료 화면, 댓글, 커뮤니티 알림, 온보딩, 에러 메시지
- **금지**: 견적, 계약, 에스크로, 결제, 분쟁, 본인인증, 개인정보, 정산

돈이 오가는 화면은 항상 Trust Mode를 유지한다.

## 5. Friendly가 기본

앱 전체 분위기는 **Friendly 90% / Lounge 10%** 비율로 운영한다. Lounge 톤은
향신료처럼 부분적으로만 쓰며, 앱 전체가 B급 감성이 되지 않도록 한다.

## 6. 구현 원칙

이 가이드라인을 적용하는 작업은 **UI/UX 및 카피 개선**이며 기능 변경이 아니다.
다음은 이 가이드라인을 이유로 변경하지 않는다.

- DB / API / 에스크로 로직 / GPS / 계약 / 인증 / XP / 레벨 / 추천 알고리즘

기존 동작은 그대로 유지하고, 화면 구성·UX·카피만 이 톤 기준에 맞춰 개선한다.
(색상 값 등 브랜드 자산 자체를 바꾸는 작업은 아님 — 1번 항목 참고.)

## 7. 최종 목표

공간마켓은 신뢰 플랫폼이면서도 사람들이 오래 머물고 싶은 서비스를 목표로 한다.

- 신뢰가 필요한 곳(Trust Mode)에서는 가장 정확하게
- 사람이 머무는 곳(Friendly Mode)에서는 가장 따뜻하게
- 커뮤니티(Lounge Mode)에서는 부담 없이 웃을 수 있게

이후 화면별 카피/UX 작업은 이 문서의 Layer 분류와 원칙을 기준으로 판단한다.
