# 라운지 운영글 SEO 구조 정책 (확정)

라운지 글을 단순 커뮤니티 글이 아니라 **검색되는 콘텐츠 자산(SEO Asset)** 으로 운영한다.

## 적용 대상
- 기존 seed 운영글 20개
- 신규 seed 운영글 / 향후 운영 콘텐츠
- 업체 전문글(권장)

---

## 1. 기본 글 구조 템플릿

운영글은 다음 구조를 기본으로 작성한다. (앱의 `RichContent` 가 `##`/`###`/`-` 를 H2·H3·목록으로 렌더)

1. **SEO 제목(H1)** — 검색 키워드 포함, 클릭 가능, 과장·낚시 금지
   - 예) `30평 인테리어 비용, 어디서 차이가 날까` / `욕실 리모델링 전 꼭 확인할 5가지`
   - 앱: 상세 화면에서 제목을 `<h1>` 으로 렌더.
2. **도입(문제 공감)** 150~300자 — 체류·공감·이탈 방지
3. **H2/H3 소제목 구조** — 블로그형 정보 구조, 문단 분리, 모바일 가독성
   - 예) `왜 비용 차이가 날까` / `업체 견적에서 보는 항목` / `실제 공사에서 많이 생기는 실수` / `비용 줄이는 방법` / `체크리스트`
4. **실질 정보** — 비용 / 체크포인트 / 순서 / 실수 사례 / 팁 / 유지관리 / 업체 질문 리스트
5. **키워드 자연 포함** — 인테리어 비용, 도배 비용, 리모델링, 욕실 리모델링, 신축 입주, 인테리어 견적 등을 본문에 자연스럽게. (키워드 도배/stuffing 금지)
6. **이미지 일치** — 제목·본문·이미지 100% 주제 일치. 이미지는 장식이 아니라 콘텐츠의 일부.
   - 앱: 이미지 `alt` 는 제목 기반으로 부여.
7. **마무리 요약** — 글 끝 핵심 정리
8. **관련글 구조** — 하단에 라운지 내부 링크(같은 카테고리 인기글 우선)로 연결.
   - 앱: 상세 하단 `관련글` 섹션이 `getRelatedLoungePosts` 로 자동 노출.

---

## 2. 길이 기준
- 권장: **2,500~4,000자**
- 최소: **1,500자 이상**
- 목표는 길이가 아니라 **검색 + 체류 + 정보 밀도**.

---

## 3. CTA 정책
운영글(seed)은 **견적요청 CTA 제거 유지**. 대신 아래만 허용한다.
- 관련글 보기 (내부 링크)
- 내 지역 업체 보기 (소프트 링크)
- 업체도 팁 공유 가능
- 저장 / 좋아요 / 댓글

광고형 CTA 금지.

---

## 4. 운영 철학
라운지는 더미글(X) · 광고판(X) 이 아니라 **검색되는 인테리어 정보/경험 플랫폼(O)** 이다.
목표: 구글·네이버 검색유입이 가능한 **공간마켓 콘텐츠 자산** 구축.

## 구현 매핑 (코드 기준)
| 정책 | 구현 위치 |
|------|-----------|
| H1 제목 | `LoungePostDetailScreen` — 제목 `<h1>` |
| H2/H3·목록 렌더 | `utils/richText.jsx` `RichContent` |
| 장문 본문(min 1,500) | `021_lounge_seed_longform.sql` |
| 이미지 alt | `LoungePostDetailScreen` — `alt={post.title}` |
| 관련글 내부링크 | `getRelatedLoungePosts` + 상세 하단 `관련글` 섹션 |
| 카드 요약 미리보기 | `LoungePostCard` `plainExcerpt` |
| 메타태그(og/description) | `utils/loungeSeo.js` `buildPostMeta` + 상세 동적 갱신 |
| seed CTA 제거 / 소프트 링크 | 견적 CTA 숨김 + `내 지역 업체 보기` 소프트 링크 |
| 상호작용(조회/하트/댓글) | `incrementLoungeView`, `lounge_post_likes`, `lounge_comments` |
